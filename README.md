# BookVerse

Premium full-stack book recommendation web application built with Node.js, Express, MongoDB Atlas, JWT, bcrypt, and a vanilla HTML/CSS/JavaScript frontend.

## Run locally

1. Copy `.env.example` to `.env` and add your MongoDB Atlas URI and JWT secret.
2. Install dependencies with `npm install`.
3. Seed the database with `npm run seed`.
4. Start the app with `npm start`.

## Included

- JWT authentication with remember-me support
- User profile, wishlist, likes, recent views, and search history
- Books, authors, genres, recommendations, and admin CRUD APIs
- Premium dark glassmorphism UI with responsive layouts
